# PTA 团队天梯赛║L1-004 计算摄氏温度

## 一、题目要求

> 给定一个华氏温度*F*，本题要求编写程序，计算对应的摄氏温度*C*。计算公式：*C*=5×(*F*−32)/9。题目保证输入与输出均在整型范围内。

### 输入格式:

输入在一行中给出一个华氏温度。

### 输出格式:

在一行中按照格式“Celsius = C”输出对应的摄氏温度 C 的整数值。

### 输入样例:

```in
150
```

### 输出样例:

```out
Celsius = 65
```

## 二、解题思路

该题为最简单的根据公式数制转换，注意题目要求 *保证输入与输出均在整型范围内*。

## 三、代码

```cpp
#include <iostream>
#include <cstring>
using namespace std;

int main()
{
    int f,c;
    cin >> f;
    c = 5*(f-32)/9;
    cout << "Celsius = " << c << endl;

    return 0;
}

```

## 四、反思总结

题目有一些简单了，注意细心不要搞错输出时候的要求，以防检查测试不通过。
